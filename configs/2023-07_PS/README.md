# MOSS PS July 2023 analysis

This directory contains config files for analysis of MOSS region 2 at PS in July 2023.

Besides the usual config files, the following files can be foudn here:

- `run_alignment_moss_july.sh`: to be executed from the top level directory.
- `moss_july_t6_reg.csv`: input files for the full analysis of testbeam data via `run_analysis_batch.py` script.
