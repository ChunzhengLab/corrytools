# MOSS PS August 2023 analysis

This directory contains config files for analysis of full MOSS dataset at PS in August 2023.
More details on the testbeam can be found in usual places: ELOG, TWiki, WP3 meeting in the weeks following the testbeam.

Besides the usual config files, the following files can be foudn here:

- `run_alignment_moss_aug.sh`: to be executed from the top level directory, alignment of all HUs
- `moss_aug_*.csv`: input files for the full analysis of testbeam data via `run_analysis_batch.py` script.
