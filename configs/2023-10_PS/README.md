# babyMOSS PS October 2023 analysis

This directory contains config files for analysis of full babyMOSS dataset at PS in October 2023.
More details on the testbeam can be found in usual places: ELOG, TWiki, WP3 meeting in the weeks following the testbeam.

Besides the usual config files, the following files can be found here:

- `run_alignment_babymoss_oct.sh`: to be executed from the top level directory.
- `moss_oct_t5_psub*.csv`: input files for the full analysis of testbeam data via `run_analysis_batch.py` script.



# Important Note: 
In this testbeam only babyMOSS top was used. It was used as t5 (top unit 5), in order to be able to use the MOSS-SW that was not supporting babyMOSS at the time of the testbeam.
